#### 개론

- 자료구조 - 알고리즘
     - 항상 붙어다님
     - *자료구조 및 실습, 자료구조 및 알고리즘 등....*
     - 그런데 우리는 2학기에 걸쳐서 따로 배운다.

- 자료구조를 배우는 이유
    - 문제를 풀기 위해서(공학적 문제)
    - 문제를 풀기 위해 모아놓은 수학적/과학적 절차가 **알고리즘**

- 문제의 조건
    - 입력(input)
    - 알고리즘을 코드로 옮긴다.

- 자료구조의 장단점을 안다. 

---

#### Python vs C

- 변수에 타입이 없다. 
- 선언 후 할당한다 (C)
- 자료형별로 크기가 정해져있음(Compile-time에 체크함)
- Python: type inference
- call by reference
    - 값을 가리킨다

- code block 표기가 다르다
    - python: indent(tab)

- 증감 연산자 없음
- 대입 연산자
